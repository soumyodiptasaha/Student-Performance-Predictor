# 🎓 Student Performance Predictor

A Machine Learning web application that predicts a student's final grade using academic performance indicators.

## 🚀 Features

- Predict final student grade (G3)
- Performance classification
- Grade classification
- User-friendly Flask web interface
- Machine Learning model using Scikit-Learn

## 🛠️ Tech Stack

- Python
- Flask
- Scikit-Learn
- Pandas
- NumPy
- HTML
- CSS

## 📊 Input Features

- Study Time
- Failures
- Absences
- G1 Grade
- G2 Grade

## 📈 Output

- Predicted Final Grade (G3)
- Performance Category
- Grade Classification

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Then open:

```text
http://127.0.0.1:5000
```
